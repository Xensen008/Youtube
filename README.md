# YouTube Video downloading Application

A full-stack application that donwloaded Youtube videos, built with modern web technologies.

## Project Structure

This project is organized into two main directories:

- **client/**: Frontend application built with React
- **server/**: Backend API built with Node.js and Express

## Installation

### Prerequisites

- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)

### Setting Up the Client

```bash
# Navigate to client directory
cd client

# Install dependencies
npm install
```

### Setting Up the Server

```bash
# Navigate to server directory
cd server

# Install dependencies
npm install
```

## Running the Application

### Client

```bash
# In the client directory
npm run dev
```

The client application will be available at `http://localhost:5173` (or the port specified in your environment).

### Server

```bash
# In the server directory
nodemon server.js
```

The server will start running at `http://localhost:5000` (or the port specified in your configuration).
